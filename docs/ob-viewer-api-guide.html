개요

콘텐츠는 OnBook 뷰어 안에서 iframe으로 실행됩니다. 뷰어와의 상호작용(페이지 이동, 파일 열기 등)은 모두 ob.viewer.api를 통해 이루어집니다.

ob.viewer.api는 콘텐츠 iframe의 window.ob.viewer.api에 자동으로 설정됩니다. NW.js(앱) 환경과 웹 환경 모두 동일한 방식으로 호출하면 됩니다.

API 카테고리

  • 페이지 뷰어의 페이지를 이동하거나 현재 페이지 정보를 조회합니다.
  • 네비게이션 씬(scene) 이동 또는 외부 링크를 엽니다.
  • 시스템 로컬 파일·폴더 열기, 앱 실행, 다운로드를 수행합니다.

사용 방법

콘텐츠 HTML에 onbook.api.min.js를 포함하면 ob.viewer.api가 자동으로 준비됩니다. 별도의 초기화 없이 바로 호출하면 됩니다. 

<!-- 콘텐츠 HTML에 포함 -->
<script src="../js/ob/onbook.api.min.js"></script>
// 페이지 이동 예시
ob.viewer.api.goPageName("p010");
ob.viewer.api.goPage(8);

// 정보 조회 예시
const current = ob.viewer.api.getPageNum();
console.log("현재 페이지:", current);

goPage

ob.viewer.api.goPage( pageNumber ) 페이지

뷰어의 페이지를 지정한 번호로 이동합니다.

파라미터 타입 설명
pageNumber number 이동할 페이지 번호 
ob.viewer.api.goPage(5);  // 5페이지로 이동

goPageName

ob.viewer.api.goPageName( pageName ) 페이지

HTML 페이지 파일명(pageName)으로 뷰어 페이지를 이동합니다.

파라미터 타입 설명
pageName string 이동할 페이지 파일명 (확장자 제외) 
ob.viewer.api.goPageName("p010");
ob.viewer.api.goPageName("unit01_lesson01_p005");

goPagePrev

ob.viewer.api.goPagePrev() 페이지

현재 페이지에서 이전 페이지로 이동합니다.

ob.viewer.api.goPagePrev();

goPageNext

ob.viewer.api.goPageNext() 페이지

현재 페이지에서 다음 페이지로 이동합니다.

ob.viewer.api.goPageNext();

goPageFirst

ob.viewer.api.goPageFirst() 페이지

첫 번째 페이지로 이동합니다.

ob.viewer.api.goPageFirst();

goPageLast

ob.viewer.api.goPageLast() 페이지

마지막 페이지로 이동합니다.

ob.viewer.api.goPageLast();

getPageNum

ob.viewer.api.getPageNum() 페이지

현재 뷰어에 표시 중인 페이지 번호를 반환합니다.

반환값: number — 현재 페이지 번호 
const page = ob.viewer.api.getPageNum();
console.log("현재:", page);

getPageNumFirst

ob.viewer.api.getPageNumFirst() 페이지

이 콘텐츠의 시작 페이지 번호를 반환합니다.

반환값: number — 시작 페이지 번호 
const first = ob.viewer.api.getPageNumFirst();

getPageNumLast

ob.viewer.api.getPageNumLast() 페이지

이 콘텐츠의 마지막 페이지 번호를 반환합니다.

반환값: number — 마지막 페이지 번호 
const last = ob.viewer.api.getPageNumLast();

getPageTotal

ob.viewer.api.getPageTotal() 페이지

전체 페이지 수를 반환합니다.

반환값: number — 전체 페이지 수 
const total = ob.viewer.api.getPageTotal();
console.log(`${ob.viewer.api.getPageNum()} / ${total}`);

historyBack

ob.viewer.api.historyBack() 네비게이션

직전에 열었던 씬(scene)으로 돌아갑니다. 자료실·팝업 모듈의 닫기 버튼에 사용합니다. 

// 닫기 버튼 클릭 시
ob.viewer.api.historyBack();

openUrl

ob.viewer.api.openUrl( url, target, options ) 네비게이션

뷰어 씬(scene) URL로 이동하거나 새 창을 엽니다. 다른 단원으로 이동하거나 팝업 창을 열 때 사용합니다.

파라미터 타입 설명
url string 이동할 씬 URL
target string "_top" · "_blank" · 창 이름
options object (선택) 창 옵션.
maximize: true — USB 버전·PC 설치 버전(NW.js 앱)에서 새 창을 최대화하여 엽니다.

url 쿼리 파라미터

씬 URL에 다음 쿼리 파라미터를 붙이면 콘텐츠가 열릴 때 특정 페이지로 바로 진입합니다.

파라미터 타입 설명
pageName string 진입할 페이지 파일명 (확장자 제외). goPageName()과 동일한 값을 사용합니다.
page number 진입할 페이지 번호. goPage()와 동일한 값을 사용합니다. 
// 다른 단원 콘텐츠로 이동 (pageName으로 진입 페이지 지정)
ob.viewer.api.openUrl(
  "/scene/contents/index.html?contentUrl=../../onbook/contents/unit01/lesson02&pageName=p010",
  "_top"
);

// page 번호로 진입 페이지 지정
ob.viewer.api.openUrl(
  "/scene/contents/index.html?contentUrl=../../onbook/contents/unit01/lesson02&page=5",
  "_top"
);

// ebook 뷰어로 이동 (page 번호로 진입 페이지 지정)
ob.viewer.api.openUrl(
  "/scene/ebook/index.html?contentUrl=../../onbook/ebook/unit01/&page=5",
  "_top"
);

// 새 창(최대화)으로 열기
ob.viewer.api.openUrl(url, "_blank", { maximize: true });

openFolder

ob.viewer.api.openFolder( path ) 시스템

지정한 경로의 폴더를 탐색기(파일 관리자)로 엽니다. NW.js(앱) 환경에서만 동작합니다.

파라미터 타입 설명
path string 열 폴더의 절대 경로 
ob.viewer.api.openFolder("C:/data/unit01/files");

openApp

ob.viewer.api.openApp( path ) 시스템

지정한 경로의 실행 파일(.exe 등)을 실행합니다. NW.js(앱) 환경에서만 동작합니다.

파라미터 타입 설명
path string 실행할 파일의 절대 경로 
ob.viewer.api.openApp("C:/data/unit01/app.exe");

downloadFile

ob.viewer.api.downloadFile( path ) 시스템

지정한 경로의 파일을 다운로드합니다. NW.js(앱) 환경에서만 동작합니다.

파라미터 타입 설명
path string 다운로드할 파일의 절대 경로 
ob.viewer.api.downloadFile("C:/data/unit01/document.hwp");

goPageLast() 페이지 마지막 페이지로 이동

getPageNumLast() 페이지 마지막 페이지 번호 반환
getPageTotal() 페이지 전체 페이지 수 반환
historyBack() 네비게이션 이전 씬으로 복귀
openUrl(url, target, opts) 네비게이션 씬 URL 이동 / 창 열기
openFolder(path) 시스템 탐색기로 폴더 열기
openApp(path) 시스템 실행 파일 실행
downloadFile(path) 시스템 파일 다운로드
