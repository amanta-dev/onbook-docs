# 프로젝트 공통 규칙

## 제품명 표기

| 표기            | 설명                                                                 |
| --------------- | -------------------------------------------------------------------- |
| `OnBook`        | 제품명. 반드시 대소문자 구분하여 표기                                |
| `OnBook 뷰어`   | 한국어 문서에서의 뷰어 명칭                                          |
| `OnBook viewer` | 영문 문서에서의 뷰어 명칭                                            |
| `onbook_viewer` | 폴더명·파일명·패키지명 등 코드상 식별자는 소문자 스네이크케이스 사용 |

> ❌ `onbook 뷰어`, `ONBOOK 뷰어`, `onBook 뷰어` — 잘못된 표기
